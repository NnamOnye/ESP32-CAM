Steps for Setup
Obtain a CA-Signed Certificate:

Use Let's Encrypt or another CA to generate your SSL certificate and private key.
Replace serverCert and privateKey with your certificate and key.
Flash the Code:

Upload the code to the ESP32 CAM using the Arduino IDE.
Access the HTTPS Server:

Access the ESP32 CAM feed via https://<ESP32-IP>.
This code ensures secure, authenticated, and encrypted access to your ESP32 CAM feed.
